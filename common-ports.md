# Common Ports

## FTP
Ports 20 and 21 are commonly used for File Transfer Protocol (FTP).

## SSH
Port 22 is used for Secure Shell (SSH), which allows secure remote login.

## Telnet
Port 23 is used for Telnet, an older remote access protocol.

## SMTP
Port 25 is used for Simple Mail Transfer Protocol (SMTP).

## DNS
Port 53 is used for Domain Name System (DNS) queries.

## HTTP
Port 80 is commonly used for standard web traffic.

## HTTPS
Port 443 is used for encrypted web traffic over TLS/SSL.

## RDP
Port 3389 is used for Remote Desktop Protocol (RDP).
